# Pages

This directory contains the main page components corresponding to application routes.

## Files

### `Index.tsx`
The main landing page of the application (Home route `/`).
-   Displays the Hero section.
-   Features the search bar and category filters.
-   Renders the resource grid.
-   Includes `ProgressTracker` and `FeaturedSection`.

### `Glossary.tsx`
The Glossary page (route `/glossary`).
-   Lists technical terms and definitions.
-   Provides search and sort functionality.
-   Features a dedicated Hero section and filter bar.

### `NotFound.tsx`
The 404 page (route `*`).
-   Displayed when the user navigates to a non-existent route.
-   Includes a link back to the Home page.
